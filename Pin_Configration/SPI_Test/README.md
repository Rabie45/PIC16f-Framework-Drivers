# PIC16f877a-Framework
 This is my own Framework for PIC16F877A microcontroll 
 
 ## Workflow:

  - Use pinSetup to setup the port as output
     - Parameters (PORT_Name ,PORT_Number ,STATE)

  - Use digitalPinWrite
     - Parameters (PORT_Name ,PORT_Number ,STATE)
