# PWM Driver PIC16F877A 
 
## Function:

 - void PWM1_Init();
   - Intialize preph. 1 for pulse width modulation
 - void PWM2_Init();
   - Intialize preph. 2 for pulse width modulation
 - void PWM1Write(uint16_t);
   - Write to preph. 1 value 

 - void PWM2Write(uint16_t);
   - Write to preph. 2 value
